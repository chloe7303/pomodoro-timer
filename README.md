# pomodoro

## 使用技術

* Vue.js
* Vue CLI
* Vuetify (UI Framework)
* Node.js
* Sass

## 功能

1. 選擇計時項目（25min工作時間/5min休息時間/15min休息時間）
2. 自訂計時時間
3. 開始計時、暫停計時、重新計時
4. 新增待辦清單
5. 編輯、刪除事項
6. 刪除全部事項、刪除已完成事項
7. 將全部事項標記已完成、將全部事項標記未完成
8. 顯示全部事項、顯示已完成事項、顯示未完成事項
9. 將記錄存進 localStroage
10. 時間到時會響鈴
